attachPDF
----------


PREREQUISITES

- KDE / Dolphin
- kdialog
- pdftk




HOWTO

In order to use this script do the following:

1) run the install.sh script to copy the files to the necessary locations

2) wait a short while until KDE/Dolphin picks up the new service menu entry

3) select one or more PDFs in dolphin, right-click and chose
   "Attach file to PDF" from the actions menus

4) You will then be prompted to select the file you want to attach to your PDF

5) After that you will be prompted to enter the page number where it should be added
