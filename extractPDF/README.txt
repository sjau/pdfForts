rotatePDF
--------


PREREQUISITES

- KDE / Dolphin
- kdialog
- pdftk
- bash
- basename (part of coreutils)



HOWTO

In order to use this script do the following:

1) run the install.sh script to copy the files to the necessary locations

2) wait a short while until KDE/Dolphin picks up the new service menu entry

3) select one PDF file in dolphin, right-click and chose
   "Extract pages from PDF" from the actions menus

4) you will then be prompted to supply what pages you want to extract

5) finally you'll be prompted to supply the file name to save the new file
